# ARAFIMS Hostel Network

A management system for ARAFIMS Hostel Network, built with Next.js, Prisma, and PostgreSQL.

## Features
- Student and staff management
- Hostel, block, room, and bedspace assignment
- Two-tiered student registration (admin and self-completion)
- Notice board and messaging
- CSV import and bulk operations
- Role-based dashboard

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your `.env` file (see `.env.example`)
4. Run database migrations: `npx prisma db push`
5. Start the development server: `npm run dev`

## Contributing
Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.
